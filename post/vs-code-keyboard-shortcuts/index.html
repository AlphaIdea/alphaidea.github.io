VS Code 快捷键

macOS 功能描述
Delete 删除光标后面的字符
Fn + F5 启动当前项目
⇧ + Fn + F12 引用跳转
⌃ + g 当前文件行跳转
⌃ + ⇥ 当前打开所有文件列表
⌃ + ` 打开终端
⇧ + ⌃ + ` 新建终端
⌃ + - 回到上一次光标所在位置
⇧ + ⌃ + - 跳转至下一次光标所在位置
⌥ + ⌫ 删除光标左侧单词颗粒度的文本
⌥ + Delete 删除光标右侧单词颗粒度的文本
⇧ + ⌥ + a 注释当前所选文本
⇧ + ⌥ + i 在选中的多行文本末尾创建光标
⌥ + ←/→ 光标移动的颗粒度就变成了单词
⇧ + ⌥ + ←/→ 选中光标前后的单词颗粒度的文本
⌥ + ↑/↓ 将当前行上/下移
⇧ + ⌥ + ↑/↓ 将当前行复制到上/下一行
⌘ + \ 拆分编辑器
⌘ + / 注释光标所在行
⌘ + b 打开/关闭侧边栏
⌘ + c 复制光标所在行
⌘ + j 打开/关闭面板
⌘ + p 搜索项目内文件
⌘ + x 剪切光标所在行
⌘ + ⌫ 删除当前行光标左侧的文本
⌘ + Delete 删除当前行光标右侧的文本
⇧ + ⌘ + x 管理扩展
⌥ + ⌘ + [ 折叠光标所处的最内层的、可以被折叠的代码
⌥ + ⌘ + ] 展开光标所处的最内层的、可以被折叠的代码
⇧ + ⌘ + d 启动和调试
⇧ + ⌘ + e 文件资源管理器
⇧ + ⌘ + k 删除光标所在行
⇧ + ⌘ + o 当前文件符号跳转
⇧ + ⌘ + p 命令面板（查找、运行命令）
⌘ + ↩︎ 在当前行下面新开始一行
⇧ + ⌘ + ↩︎ 在当前行上面新开始一行
⌘ + ←/→ 将光标移动到行首或行末
⇧ + ⌘ + ←/→ 选中光标所在位置到行首或行末的文本
⌘ + ↑/↓ 将光标移动到文档的第一行或最后一行
⌥ + ⌘ + ↑/↓ 创建多光标
⇧ + ⌘ + ↑/↓ 选中光标所在位置到文档第一行或最后一行
⌥ + ⌘ + f 打开替换窗口
⇧ + ⌘ + f 跨文件搜索
⌘ + k + ⌘ + 0 折叠所以可折叠代码
⌘ + k + ⌘ + j 展开所以可展开代码
